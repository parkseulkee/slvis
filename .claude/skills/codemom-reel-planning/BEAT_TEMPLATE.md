# 비트 시트 템플릿

새 릴스 기획할 때 이 표를 복사해서 채웁니다.
레퍼런스: `posts/post-04-claude-design-system/reel/VOICEOVER.md`

---

## 기본 규칙

| 항목        | 스펙                                          |
| ----------- | --------------------------------------------- |
| 총 비트 수  | 5~7개                                         |
| 총 길이     | 20~30초                                       |
| Beat 01     | 훅 — 첫 1초 안에 키워드 노출                  |
| Beat 마지막 | 아웃트로 — 자막 없음, 핸들(@codemom.life)만   |
| 비트당 길이 | 2~5초 (훅 3초, 정보 비트 3~4초, 아웃트로 2초) |

---

## 비트 시트 (복사해서 사용)

```markdown
| #   | 시간          | 화면 | 보이스오버 | 자막 | 전환        |
| --- | ------------- | ---- | ---------- | ---- | ----------- |
| 01  | 0:00 → 0:\_\_ |      |            |      | 페이드 인   |
| 02  | 0:** → 0:**   |      |            |      | 즉시 컷     |
| 03  | 0:** → 0:**   |      |            |      | 페이드      |
| 04  | 0:** → 0:**   |      |            |      | 슬라이드 업 |
| 05  | 0:** → 0:**   |      |            |      | 즉시 컷     |
| 06  | 0:** → 0:**   |      |            |      | 페이드      |
| 07  | 0:** → 0:**   |      |            |      | 페이드 아웃 |
```

---

## 전환 선택 기준

| 전환        | 언제                                |
| ----------- | ----------------------------------- |
| 페이드 인   | 첫 비트 시작                        |
| 즉시 컷     | 리듬 강조, 대비 효과                |
| 페이드      | 분위기 전환, 호흡 주기              |
| 슬라이드 업 | 정보 추가 느낌 (타입·리스트 비트에) |
| 페이드 아웃 | 마지막 비트                         |

---

## VOICEOVER.md 출력 형식

비트 시트 확정 후 이 형식으로 저장합니다:

```markdown
# 릴스 보이스오버 대본 — [post 제목]

**@codemom.life · 약 \_\_초 · [톤 설명]**

> [한 줄 기획 의도]

---

## 보이스오버 + 화면 구성

| 시간        | 말 (보이스오버) | 화면        | 자막          |
| ----------- | --------------- | ----------- | ------------- |
| 0:00~0:\_\_ | "..."           | [화면 설명] | [자막 텍스트] |

...
| 0:**~0:** | "저장해두세요." | @codemom.life 아웃트로 | — |

---

## 녹화 가이드

- **Hook:** [어떻게 찍을지]
- **나머지 비트:** [화면 녹화 or 앱 화면 or 기존 PNG]
- **목소리 톤:** [빠름/보통, 특이사항]
- **음악:** CapCut 오디오 라이브러리 → `[검색어]`, 볼륨 \_\_\_%

---

## CapCut 자막 스펙

- 폰트: Pretendard Bold
- 크기: 56~64pt
- 색: 흰색 #FFFFFF 또는 크림 #FAF7F2
- 위치: 하단 중앙, 안전영역 안
- 아웃트로 비트: 자막 없음
```
