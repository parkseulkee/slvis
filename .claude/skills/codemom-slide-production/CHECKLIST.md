# Pre-Publish Checklist

업로드 버튼 누르기 전 마지막 점검. 사용자와 함께 한 줄씩 체크하세요.

---

## 📐 스펙

- [ ] PNG 8장 전부 **1080×1350** 또는 **2160×2700** (2x)
- [ ] 파일명이 슬라이드 순서대로 정렬됨 (`01-` ~ `08-`)
- [ ] 파일 사이즈 각 **8MB 이하** (인스타 리밋)
- [ ] JPEG 아닌 **PNG** (투명/그라디언트 보존)

## 🎨 브랜드

- [ ] 모든 슬라이드 상단에 **mono 라벨** (`DEV · D3`, `APP · JANJAN` 등)
- [ ] 모든 슬라이드 하단에 **`@codemom.life  ·  SWIPE →`** (마지막은 `THANK YOU` 또는 `END`)
- [ ] 페이지 카운터 `NN / total` 존재
- [ ] 악센트 컬러 **하나만** 사용 (혼합 포스트는 예외 허용)
- [ ] 모드 전환 **최대 1회** (LIFE↔TECH)

## ✍️ 카피

- [ ] 이모지 **0개**
- [ ] ㅋㅋ / ㄷㄷ / ㅠㅠ **0개**
- [ ] "PLOT TWIST", "반전!", "⚡" 같은 자기선언 **0개**
- [ ] 한 슬라이드 최대 **두 문장**
- [ ] 코드/도구명은 **모노 폰트**로 렌더됨
- [ ] 최소 폰트 **24px** (1080×1350 기준)

## 📱 인스타 전용

- [ ] 캡션 초안 준비됨 (포스트 설명)
- [ ] 해시태그 **5~8개** (핵심: `#코드맘`)
- [ ] 지역/위치 태그 안 넣기 (페르소나 불일치)
- [ ] 첫 슬라이드에 카피 잘림 없는지 **피드 미리보기**로 확인 (`ui_kits/instagram/`)
- [ ] 알트 텍스트 (접근성): 각 슬라이드 한 줄 설명 준비

## 🎬 릴스 (해당 시)

- [ ] 1080×1920
- [ ] 20초 이하
- [ ] 첫 1초에 훅 노출
- [ ] 자막 번인 불필요한지 체크 (음소거 시청자용)

## 🔄 보관

- [ ] `posts/post-NN-<slug>/exports/png/` 커밋됨
- [ ] `plan.md`의 `status:` 를 `published` 로 업데이트
- [ ] 발행일 · 최종 반응 메모 남기기 (다음 기획 참고용)

---

## 발행 후 회고 (일주일 뒤)

- [ ] 어느 슬라이드에서 이탈이 많았나 (인사이트 확인)
- [ ] 저장 · 공유 수 기록
- [ ] 댓글에 반복 질문이 있으면 다음 포스트 소재로 기록
