## CapCut 편집 순서

### 1. 새 프로젝트

- 비율 **9:16** → 해상도 **1080×1920**

### 2. 클립 추가

| 클립 | 소스                             | 지속시간 |
| ---- | -------------------------------- | -------- |
| 01   | Claude Design 녹화 영상          | 3.0초    |
| 02   | 취소선 화면 녹화                 | 3.0초    |
| 03   | 팔레트 화면 녹화                 | 4.0초    |
| 04   | 타입 화면 녹화                   | 4.0초    |
| 05   | 템플릿 그리드 화면 녹화          | 4.0초    |
| 06   | post-01 + post-02 화면 녹화      | 5.0초    |
| 07   | @codemom.life 아웃트로 화면 녹화 | 2.0초    |

### 3. 전환 설정

- 01→02: 없음 (즉시 컷)
- 02→03: 페이드 0.3초
- 03→04: 슬라이드 업 0.4초
- 04→05: 없음 (즉시 컷)
- 05→06: 페이드 0.4초
- 06→07: 페이드 아웃 0.5초

### 4. 보이스오버 추가

- CapCut → 오디오 → 녹음
- `VOICEOVER.md`의 대본 읽기
- 빠른 속도, 마침표마다 0.2초 쉼

### 5. 자막 추가 (`CAPTIONS.srt` 임포트 또는 수동)

- 폰트: **Pretendard Bold** (없으면 Bold Gothic)
- 크기: 56~64pt
- 색: 흰색 `#FFFFFF` 또는 크림 `#FAF7F2`
- 위치: 하단 중앙, 안전영역 안
- 비트 07 아웃트로: 자막 없음

### 6. 음악

- CapCut 오디오 라이브러리 → `로파이` / `집중` 검색
- 볼륨 50% (보이스오버가 주 전달 수단)
- 시작 0초, 영상과 함께 페이드 아웃

### 7. 내보내기

- 해상도: **1080p**
- 프레임: **30fps**
- 비트레이트: **높음 (High)**
