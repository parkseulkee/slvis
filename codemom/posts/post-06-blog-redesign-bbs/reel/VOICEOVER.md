## 녹화 가이드

- **Beat 01 (Hook):** 마우스를 **느리게** 좌→우→3초 정지. 빠르게 휘저으면 슬비스 시선 추적이 안 보임. 캔버스 영역(슬비스 360px)이 화면의 60% 이상 차지하도록 줌인 — 픽셀 디테일이 보여야 "살아있는" 느낌 강해짐.
- **Beat 02:** [1]→[2]→[3] 키 누를 때 페이지 전환되는 짧은 컷 모음. 각 키 사이 약 1초 간격. 키캡 클로즈업은 따로 안 찍어도 됨 — 자막에서 처리.
- **Beat 03:** 옛 Jekyll 블로그 BEFORE 스크린샷 필요. 없으면 Wayback Machine 또는 git history에서 복원. CapCut에서 좌(BEFORE) / 우(AFTER) 분할 화면.
- **Beat 04~05:** Claude Design 채팅 스크린샷 + Claude Code 작업 흔적. 화면 녹화 짧게(2~3초) 만들어 슬라이드 업 전환에 얹기.
- **Beat 06:** 블로그 홈 띄워둔 상태로 키보드·마우스 4초 안 만지면 타이틀바 우측 "감도 高"가 "低"로 자동 전환됨. 이 변화를 클로즈업으로 잡으면 한 컷이 됨.
- **목소리 톤:** 빠른 속도, 마침표마다 0.15초 쉼. 호흡 짧게. "여러분~" 호칭은 안 씀 — 이번 릴스는 비주얼이 끌고 가는 구조라 차분한 개발자 톤이 더 어울림.
- **음악:** CapCut 오디오 라이브러리 → `lo-fi terminal` / `chiptune` / `synthwave` 검색. 볼륨 35~40% (보이스오버 가리지 않게). 인스타 업로드 후 앱에서 직접 추가하는 것도 옵션.

- 위치: 하단 중앙, 안전영역 안 (하단에서 약 220px 위)
