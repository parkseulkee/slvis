## 3. 보이스오버 추가

1. CapCut 우측 패널 → 오디오 → 녹음
2. VOICEOVER.md 표 그대로 비트별로 분리 녹음 (시간 맞추기 쉬움)
3. 각 비트 클립 위에 정렬 — 비트 내 보이스오버는 비트 시작 후 0.2초 뒤에 시작
4. 톤: 빠른 속도, 마침표마다 0.15초 쉼

---

## 4. 자막 추가 (CAPTIONS.srt)

1. CapCut 텍스트 → 자막 → SRT 임포트 → `CAPTIONS.srt` 선택
2. 폰트/색/위치 일괄 적용 (Pretendard Bold 60pt, 흰색, 하단 중앙)
3. Beat 07 자막 없음 — SRT에도 빠져있음 확인
4. `[1][2][3]` `Claude Design` `Astro` 같은 영문/숫자 자막은 JetBrains Mono로 폰트만 교체 검토

## 5. 음악 추가

**옵션 A — 인스타 트렌드 음악 (권장)**

- 편집 단계에서는 음악 추가하지 않음
- mp4 export 후 인스타 업로드 → 앱 내 음악 라이브러리에서 트렌드 곡 선택
- 보이스오버 볼륨 100%, 음악 자동으로 30~40% 깔림

**옵션 B — CapCut 음악 번인**

- CapCut 오디오 라이브러리 → `lo-fi terminal` / `chiptune` / `synthwave` 검색
- 볼륨 35~40% (보이스오버 가리지 않게)
