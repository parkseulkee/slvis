슬비스

Obsidian + AI로 만드는 개인 비서

by 박슬기 · 2026.03

왜 만들었나

  • 육아 중 기록할 것이 너무 많다
  • 일정, 수유, SNS 글감, 투자, 아티클… 흩어진 정보
  • AI가 내 기록을 읽고, 정리하고, 연결해주면?
  • 대화는 휘발되지만, 기록은 남는다

핵심 컨셉

  • AI 도구에 종속되지 않는다 — 도구는 바뀌어도 기록은 Obsidian에 남는다
  • 함께 성장하는 비서 — 처음부터 완벽하지 않다. 피드백으로 진화한다
  • 기록 중심 — 모든 지식, 선호, 회고를 Obsidian에 저장

기술 구조

슬비스 (Claude Code)
├── CLAUDE.md         ← 행동 규칙
├── Obsidian Vault    ← 지식 베이스 (두뇌)
│   └── Obsidian CLI  ← 읽기/쓰기 도구
├── gcalcli           ← Google 캘린더
└── gh CLI            ← GitHub

Vault 구조

폴더용도
Daily일일 노트 (일정/할일/뉴스/기록)
육아겨울이 성장 기록, 수유
SNS블로그/인스타 글감
투자ETF 분석, 투자 메모
아티클읽을거리 추천, 요약
슬비스프로젝트 기록, 운영 가이드, 회고

슬비스 폴더

AI 도구가 바뀌어도 유지되는 지식

  • 사용자 프로필 — 내가 어떤 사람인지
  • 운영 가이드 — 내가 선호하는 방식
  • 프로젝트 정보 — 슬비스의 구조와 원칙
  • 회고 — 슬비스의 실수와 개선 기록

Skill: /daily

매일 아침 한 번의 명령으로:

  1. 오늘 일일 노트 + Google 캘린더 일정 확인
  2. 긱뉴스에서 AI/기술 뉴스 큐레이션
  3. 경제/정치 주요 뉴스 큐레이션
  4. 결과를 일일 노트에 자동 저장

다른 skill은 필요에 따라 추가 예정

실행 결과 예시

📅 일정
  ❤️ 가족 | 출산 휴가 (진행 중)

📰 AI/기술
  1. AI 에이전트 워크플로우 3가지 패턴
  2. OpenAI 에이전트 구축 실용 가이드
  3. Claude 인터랙티브 시각 자료 생성

🏛️ 경제/정치
  1. 석유 최고가격제 시행
  2. 3월 상순 수출 역대 최대
  3. 대통령 지지율 66%

→ 일일 노트에 자동 저장

기록 원칙

저장소용도
Obsidian vault모든 지식 · 기록 · 사용자 선호
AI 도구 내부현재 도구의 동작 설정만 (최소한)

원칙: AI 도구는 언제든 바뀔 수 있다.
중요한 기록은 반드시 Obsidian에 남긴다.

사용 예시

  • 📝 "오늘 겨울이 수유 기록해줘" → 육아 노트에 저장
  • 📰 /daily → 일정 + 뉴스 브리핑
  • 💡 "이거 SNS에 올릴만하지 않아?" → SNS 글감 생성
  • 📅 "다음주 일정 알려줘" → gcalcli로 조회

앞으로

  • 더 많은 skill 추가 (/task, /article, /review ...)
  • 수면교육 시작 시 수면 기록 기능
  • SNS 글감 자동 추출
  • 투자 포트폴리오 추적
  • 슬비스와 함께 성장하는 과정을 기록하고 공유
